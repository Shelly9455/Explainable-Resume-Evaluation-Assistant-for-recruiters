# Explainable Resume Evaluation Assistant

## Problem

Recruiters increasingly use AI for resume screening but often struggle to trust AI recommendations because:
- Reasoning is hidden
- Evidence is unclear
- Recommendations are difficult to verify
- Important requirements may be overlooked

## Solution

An AI-powered Explainable Resume Evaluation Assistant that:

1. Analyzes Job Descriptions
2. Generates a structured hiring rubric
3. Requires recruiter approval of the rubric
4. Evaluates resumes only against approved criteria
5. Provides evidence-based reasoning
6. Exposes assumptions and uncertainties

## Key Features

- JD Analysis
- Rubric Generation
- Requirement Coverage Analysis
- Explainable Scoring
- Evidence-Based Evaluation
- Recruiter Approval Workflow

## Product Workflow

JD Analysis → Rubric Finalization → Resume Evaluation
